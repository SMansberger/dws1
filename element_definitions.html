HTML Element Definitions

<h1>
A level one header.
<h2>
A level two header.
<h3>
A level three header.
<h4>
A level four header.
<h5>
A level five header.
<h6>
A level six header.
<p>
Create a new paragraph (text).
<b>
Create bold text.
<i>
Create italicized text.
<sup>
Display text as a superscript.
<sub>
Display text as a subscript
<br />
Used to create a line break inside a paragraph. Used inside the <p> tag.
<hr />
Also used inside a <p> tag, <hr /> creates a break used between themes by creating a horizontal line between them.
<strong>
Used to show an element has strong importance. Default browser behavior is do display <strong> contents as bold.
<em>
Used to display emphasis on an item. Browsers will display tagged content in italic.
<blockquote>
Used to display quotes long enough to take up an entire paragraph.
<q>
Used to display quotes around text. (IE does not do this.)
<abbr>
Used when an abbreviation or acronym is used.
Usage:<abbr title ="Boulevard">blvd.<abbr>
<cite>
Used to indicate where citation is used when referencing a piece of work for research.
<dfn>
Used to indicate that an item that is being defined.
<address>
This is used to contain details pertaining to the author of a page.
<ins>
Used to show when an element has been inserted into text.
<del>
Used to show when an element has been removed or deleted from a text.
<s>
Used to show when information is no longer accurate. (browsers display this by striking a line through tagged text.
<ol>
Used to create an ordered (numbered) list. Items inside the list are displayed using the <li> tag. (Multiple <ol> tags can be embedded to number lists inside of numbered lists.)
<ul>
Used to create an unnumbered list; items are displayed with bullets and are marked with <li> tags.
<dl>
Used to create a list of definitions.
<dt>
Used to display the items being defined within a <dl> list.
<dd>
Used to describe the item being defined; the item(s) definition resides here.
